# Shared

Código reutilizável e transversal, que **não depende** de `src/features/**`.

## Subpastas sugeridas
- `ui/`: primitives (Modal, Label, Spinner, etc.)
- `components/`: componentes genéricos (layout base, wrappers)
- `hooks/`: hooks genéricos
- `api/`: cliente HTTP/fetch wrappers comuns
- `lib/`: utilitários puros (date, slug, errors)
- `seo/`, `i18n/`: helpers compartilhados
